# Test Data Files

This directory contains test case specifications for benchmarking the Employee Scheduling optimization algorithms.

## Test Cases

### SMALL (small-dataset.txt)
- **Employees:** 5
- **Shifts per day:** 3
- **Days:** 3
- **Total shifts:** 9
- **Purpose:** Quick validation and testing
- **Expected solve time:** < 1 second
- **Expected score:** 0hard / ~15soft

### MEDIUM (medium-dataset.txt)
- **Employees:** 20
- **Shifts per day:** 15
- **Days:** 5
- **Total shifts:** 75
- **Purpose:** Realistic scenario testing
- **Expected solve time:** 5-10 seconds
- **Expected score:** 0hard / ~50soft

### LARGE (large-dataset.txt)
- **Employees:** 50
- **Shifts per day:** 30
- **Days:** 7
- **Total shifts:** 210
- **Purpose:** Stress testing and performance evaluation
- **Expected solve time:** 20-30 seconds
- **Expected score:** 0hard / ~120soft

## Usage

These test cases are automatically generated by `DemoDataGenerator.generateConfigurableDataSet()` using the parameters specified above.

To run benchmarks with these test cases:

```bash
mvn clean compile -Dmaven.test.skip=true
mvn exec:java -Dexec.mainClass="org.acme.employeescheduling.rest.BenchmarkRunner"
```

Results will be saved to: `target/benchmarks/`
